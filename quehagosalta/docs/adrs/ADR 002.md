Fecha: 02/06/2026

Estado: Aceptado

Decision: La eleccion del primer hito para avanzar en el desarrollo de la aplicación

Objetivo: Tener la app conectada a Supabase y renderizando el mapa base centrado en Salta.

Tareas: 
    Tareas de Backend (Supabase/Postgres):
        Crear el proyecto en Supabase.
        Activar la extensión PostGIS.
        Crear la tabla business_category (Id, name, icono_key").
    Tareas de Frontend (Flutter):
        Inicializar el proyecto Flutter y configurar el paquete supabase_flutter.
        Integrar flutter_map (OpenStreetMap) y configurar el paquete geolocator para pedir permisos y obtener el GPS del usuario.
        
Resultado: El usuario abre la app, se detecta su ubicación y ve el mapa base de Salta.
